    에이치 앤 티(주) 웹 사이트 제작

    PROJECT OVERVIEW

    에이치 앤 티는 하드웨어, 소프트웨어, 서버 판매 및 유지보수 전문업체로서 최적화된 서비스를 제공하는 회사입니다. 제품의 구매와 렌탈 서비스가 부각되게 제작하였습니다.

    TYPE

    MAIN FEATURES

    • 반응형 웹 사이트

      PC부터 Tablet, Mobile까지 각각의 디바이스에 맞추어 적용되는 반응형 웹 사이트 입니다.

    • 실사용 제품 활용

      제품판매 및 렌탈의 특색을 살려 다양한 기기들을 이미지로 사용함으로써 고객들이 회사의 서비스 이용에 좀더 신뢰성을 가질수 있도록 디자인 하였습니다.

    • 직접 선택하는 사양

      제품 구매 및 렌탈 서비스를 이용할 때 기본 사양 외 직접 작성할 수 있도록 하였습니다.

    실제 제품을 사용하여
    신뢰를 유지하는 디자인

    회사에서 판매 및 대여하는 제품의 실제이미지를 사용하여 회사에 대한 신뢰감을 유지하며
    제공하는 정보와 제품이 동일하다는 것을 고객에게 전달할 수 있습니다.

    내가 직접 선택가능한 옵션

    기존에 제공하는 사양뿐만 아니라
    원하는 사양을 스스로 적어 제출할 수 있게 설정하였습니다.
