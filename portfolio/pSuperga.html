    수페르가 리뉴얼 프로젝트

    PROJECT OVERVIEW

    수페르가는 이탈리아의 명품 신발 브랜드로써 많은 이들에게 사랑을 받는 브랜드입니다. 작업 목적은 기존 홈페이지의 노후 및 업데이트 부재로 방문이 적은 홈페이지를 개선을 위한 리뉴얼 프로젝트입니다.

    TYPE
    Web, Mobile Web
    PERIOD
    2016년 02월 ~ 06월
    MEMBER
    1인 프로젝트
    POSITION
    디자인 / 퍼블리싱
    TOOL
    HTML / CSS / jQuery
    사이트 바로가기

    MAIN FEATURES

    신발 의류 판매 브랜드인 수페르가의 고객들은 남녀노소할거없이 전 연령대입니다. 그 중 소비가 가장 많이 일어나는 10 ~ 20대 고객층을 타겟으로 정하여 홈페이지를 제작하였습니다. 또한 모델 '이솜'의 화보를 사용하여 구매욕구를 증가시킬수 있게 하였습니다.

    • 남성 & 여성

    • 10대 ~ 20대

    • 모델 '이솜'

    온라인 스토어 형식의 홈페이지의
    초기 설정

    수페르가의 스케치와 목업작업시 중점으로 둔것은 온라인 스토어형식이면서도 브랜드의 소개와 역사를 빼지않는 것이였습니다.
    그리고 모바일로 보여질때의 메뉴의 이동 및 배너의 위치변경 등을 고려하여 작업하였습니다.

    수페르가 모바일 웹

    반응형으로 제작된 수페르가 리뉴얼 사이트는 각종 디바이스의 맞추어
    PC뿐만 아니라 Tablet, Mobile에서도 원활하게 사용이 가능합니다.
