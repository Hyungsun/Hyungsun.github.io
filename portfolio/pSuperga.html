    수페르가 리뉴얼 프로젝트

    PROJECT OVERVIEW

    수페르가는 이탈리아의 명품 신발 브랜드로써 많은 이들에게 사랑을 받는 브랜드입니다. 작업 목적은 기존 홈페이지의 노후 및 업데이트 부재로 방문이 적은 홈페이지를 개선을 위한 리뉴얼 프로젝트입니다.

    TYPE
    Web, Mobile Web
    PERIOD
    2016년 02월 ~ 06월
    MEMBER
    개인 작업
    POSITION
    디자인 / 퍼블리싱
    TOOL
    HTML / CSS / jQuery
    사이트 바로가기

    MAIN FEATURES

    • 반응형 웹 사이트

      PC부터 Tablet, Mobile까지 각각의 디바이스에 맞추어 적용되는 반응형 웹 사이트 입니다.

    • 카테고리 분류

      다양한 고객층을 보유하고있는 브랜드로 성별, 연령층에 따른 분류와 모든 제품부터 특정 제품을 나누어 볼 수 있는 카테고리기능을 제공합니다.

    • 브랜드 컬러 강조

      수페르가 로고의 색상 2가지를 포인트 컬러로 지정하여 메뉴부터 버튼까지 다양한 영역에서 사용됩니다.

    온라인 스토어 형식의 홈페이지의
    초기 설정

    수페르가의 스케치와 목업작업시 중점으로 둔것은 온라인 스토어형식이면서도 브랜드의 소개와 역사를 빼지않는 것이였습니다.
    그리고 모바일로 보여질때의 메뉴의 이동 및 배너의 위치변경 등을 고려하여 작업하였습니다.

    수페르가 모바일 웹

    반응형으로 제작된 수페르가 리뉴얼 사이트는 각종 디바이스의 맞추어
    PC뿐만 아니라 Tablet, Mobile에서도 원활하게 사용이 가능합니다.
