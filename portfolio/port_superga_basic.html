    수페르가 [SUPERGA]

    기획

    POSITION : 1인 프로젝트 디자인 / 퍼블리싱

    제작 기간 : 2016년 02월 ~ 06월

    CONCEPT : 기존 홈페이지의 리뉴얼 및 심플한 디자인

    TARGET : 10 ~ 20대의 청소년/성인 층

    TOOL : Adobe Photoshop, illustrator / HTML5, CSS3, jQuery

    People's Shoes of Italy, SUPERGA

    색상과 서체

    주조색과 강조색을 혼합하여 해당 브랜드에 맞는 색상을 유지하고 "맑은고딕", "arial" 서체를 사용하여 기본적이면서도 익숙한 느낌을 받을 수 있게 제작하였습니다.

    주조색

    #333333

    강조색

    #000E46

    보조색

    #777777

    제작 과정

    STEP1. 스케치

    작업 전 스케치 작업을 통하여 웹 사이트 전체의 흐름과 느낌, 구조를 미리 작업하여 작업을 좀 더 쉽고 구체적으로 진행하였습니다.

    • PC (W:1025)

    • TABLET (W:768)

    • MOBILE (W:480)

    STEP2. 목업

    스케치 작업을 바탕으로 일러스트 및 포토샵을 이용하여 구체적인 수치와 위치 색상을 선택하여 작업하였습니다.
